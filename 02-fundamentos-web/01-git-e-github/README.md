# Git e GitHub

Estudos e práticas sobre controle de versão com Git e hospedagem de projetos no GitHub.

## 📌 Tópicos Estudados

- Configuração inicial do Git (user.name, user.email, chaves SSH).
- Ciclo de vida dos arquivos (untracked, unmodified, modified, staged).
- Comandos essenciais: `git status`, `git add`, `git commit`, `git log`, `git show`.
- Ramificações (Branches): criação, alternância e mesclagem (`git merge`).
- Resolução de conflitos de merge.
- Uso de repositórios remotos: `git remote`, `git push`, `git pull`, `git clone`, `git fetch`.
- Boas práticas: Commits semânticos (Conventional Commits) e fluxo de trabalho Git Flow básico.
