# HTML5 & CSS3

Estudos e projetos práticos envolvendo estruturação semântica de páginas web e estilizações modernas com CSS3.

## 📌 Tópicos Estudados

- **HTML5:** Tags semânticas (`<header>`, `<nav>`, `<main>`, `<article>`, `<section>`, `<aside>`, `<footer>`), acessibilidade (WAI-ARIA) e formulários estruturados.
- **CSS3:** Box Model, Seletores avançados, Posicionamento (`static`, `relative`, `absolute`, `fixed`, `sticky`).
- **Flexbox & Grid Layout:** Layouts flexíveis e bidimensionais responsivos.
- **Responsividade:** Media Queries e abordagem Mobile-First.
- **Design Avançado:** Variáveis CSS (Custom Properties), Transições, Animações e Pseudo-classes/Pseudo-elementos.
