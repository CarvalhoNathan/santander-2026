# JavaScript ES6

Anotações e códigos práticos sobre lógica de programação no navegador e no Node.js, focando nos recursos modernos do ECMAScript 6+ (ES6+).

## 📌 Tópicos Estudados

- Variáveis (`let`, `const`), escopo e Hoisting.
- Tipos de dados e estruturas de dados dinâmicas.
- Funções tradicionais vs Arrow Functions e Closures.
- Estruturas condicionais e de repetição.
- Manipulação do DOM (Document Object Model) e escuta de eventos.
- Recursos modernos (ES6+): Destructuring, Rest/Spread Operator, Template Literals, Promises, Async/Await.
- Métodos avançados de arrays (`map`, `filter`, `reduce`, `find`, `forEach`).
- Consumo de APIs (Fetch API e tratamento de erros).
