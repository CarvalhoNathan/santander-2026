# 02 — Fundamentos Web

Este módulo é dedicado ao aprendizado dos pilares fundamentais do desenvolvimento web e controle de versão, servindo como base sólida antes de avançar para bibliotecas e frameworks de front-end.

## 📂 Submódulos

1. **[Git e GitHub](./01-git-e-github/)** — Fluxo de trabalho de controle de versão, ramificações (branches), commits, pulls/pushes e colaboração no GitHub.
2. **[HTML5 e CSS3](./02-html5-css3/)** — Estrutura semântica para web e estilização avançada usando CSS moderno (Flexbox, Grid, Variáveis CSS, responsividade).
3. **[JavaScript ES6](./03-javascript-es6/)** — Lógica de programação, manipulação do DOM, funções assíncronas, Promises, APIs, métodos modernos de array e novos recursos do ECMAScript.

---

## 🛠️ Como utilizar esta pasta

Cada submódulo está organizado da seguinte forma:
- **`anotacoes/`** — Notas teóricas tomadas durante as aulas.
- **`praticas/`** — Exercícios resolvidos e códigos de prática.
- **`projetos/`** — Pequenos projetos ou laboratórios práticos.
