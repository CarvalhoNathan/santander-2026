# React

Biblioteca desenvolvida pelo Facebook (Grupo Meta) para construção de interfaces em diferentes ambientes/dispositivos.

- Funciona em ambientes variados.
- Construção de aplicações de maneira eficiente.
- Reatividade e atualizações de interface de maneira automática.
- Peças individuais de interfaces chamadas de "componentes" que podem ser reutilizadas em diferentes aplicações.
