# 04 — Desafios

Nesta pasta ficam concentrados todos os desafios de código (algoritmos e lógica) e desafios de projeto entregues oficialmente durante o bootcamp na plataforma da DIO.

## 📂 Submódulos

1. **[Desafios de Código](./desafios-de-codigo/)** — Resolução de problemas de lógica de programação, algoritmos e testes práticos de codificação.
2. **[Desafios de Projeto](./desafios-de-projeto/)** — Projetos práticos completos propostos no bootcamp, incluindo o desenvolvimento da aplicação final de portfólio.
3. **[Prompt de Desafios Criativos](./prompt-meus-desafios-criativos.md)** — Prompt estruturado e otimizado para gerar a proposta detalhada do projeto final (Educador Financeiro Inteligente) via Inteligência Artificial.
