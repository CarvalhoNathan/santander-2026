# Desafios de Código

Resolução de testes e problemas de algoritmos propostos no bootcamp da DIO para exercitar a lógica de programação.

## 📂 Desafios Resolvidos

1. **[Desafio 01: Mapeamento de Códigos de Produtos](./01-desafio.js)** — Conversão de códigos abreviados (`CC`, `CD`, `EMP`, `INV`) para seus respectivos nomes por extenso de produtos financeiros digitais.
2. **[Desafio 02: Formatador de Saldo de Usuário](./02-desafio.js)** — Conversão de saldo recebido em centavos para número decimal e formatação para moeda brasileira (Real - `R$`) de forma correta e sem espaçamentos.
